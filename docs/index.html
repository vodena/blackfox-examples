    Black Fox

    Actively pursues the best predictive model for you

    Black Fox is strong and agile AI automation assistant capable for heavy-lifting required for neural network parameter optimization. Once in action the Black Fox relentlessly pursues the best posible predictive model for the data set at hand, and when caught the best model is readily delivered for further predictive analysis.

    Install Black Fox and dive right in

    At present, Black Fox is available as a Python 3 package, while bindings for other languages are well underway. Black Fox is designed to be as unobtrusive as possible and fits in perfectly within any data science workflow. It can be installed from PyPi using a plethora of modern Python package managers, pip, pipenv or poetry.

     $ pip install -U blackfox

    Dive in with your test and train data directly from an interactive Python session:

     >> from blackfox import BlackFox
 >> bf = BlackFox()
 >> (nn_io, nn_info, nn_md) = bf.optimize_keras_sync(
      x_train, 
      y_train
   )
